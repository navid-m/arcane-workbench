# ArcaneDB Workbench

## Keyboard Shortcuts

| Action              | Shortcut         |
|---------------------|------------------|
| New Script          | Ctrl/Cmd + N     |
| Open Script         | Ctrl/Cmd + O     |
| Save Script         | Ctrl/Cmd + S     |
| Save Script As      | Ctrl/Cmd + Shift + S |
| Run Script          | F5               |
| Run Selected Text   | Ctrl/Cmd + Enter |
| Show Bucket Graph   | Ctrl/Cmd + G     |
| Toggle DevTools     | Ctrl/Cmd + Shift + I |

## Bucket Graph

The Bucket Graph visualizes a database schema as an interactive force-directed graph, similar to Obsidian's graph view. Each bucket is represented as a node showing:

- Bucket name
- Number of fields
- Record count

Buckets with common field names and types are automatically linked, helping you understand relationships in a data model. You can drag nodes, zoom, and pan to explore a schema.
